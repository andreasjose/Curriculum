Codigo para compilar o servidor
gcc -std=c99 -o servidor.bin servidor.c

Codigo para executar o servidor
./servidor.bin

Codigo para compilar o Cliente
gcc -std=c99 -o cliente.bin cliente.c

Codigo para executar o cliente
./cliente.bin X

Obs.: X é o número da porta que o heroi deseja acessar.

O Servidor ira gerar um arquivo de registro chamado dados.txt
nele aparece
Ex:
25/Sep/2018 - 17:19:03, p5, 1458, Autorizado

Sobre o arquivo com os acessos
Precisei colocar a quantidade nele
o nome dele é cadastros.txt
Ex:
1458,Tony Stark, 5
4287,Steve Rogers, 4
4298,Thor, 2
6528,Natasha Romanoff, 3
7458,Bruce Banner, 1

Obs.: O 5 é o numero de herois cadastrados
      No exemplo que foi apresentado tem um espaço depois da primeira virgula.
      Não coloca esse espaço se não ele capitura o espaço no começo do nome.
      deixa exatamente como está codico,nome, porta.
      coloquei o arquivo com os herois cadastrados.
